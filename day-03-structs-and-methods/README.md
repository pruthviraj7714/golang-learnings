# Day 03 - Structs & Methods

## What I Learned

- Structs (similar to classes)
- Methods with receivers
- Pointer vs value receivers
- Struct composition (instead of inheritance)

## 📝 Notes

- Go uses composition instead of inheritance and prefers simplicity.
- Structs are value types
- Pointer receiver is used to modify the original struct
- Value receiver is used to read the struct
- Anonymous struct is used to create a struct without a name
