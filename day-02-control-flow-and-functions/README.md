# Day 02 - Control Flow & Functions

## What I Learned

- If-else conditions
- Switch statements (cleaner than JS)
- For loops (only loop in Go)
- Functions in Go
- Multiple return values (important concept)

## 📝 Notes

- Go functions can return multiple values, which is very useful for error handling and logic.
- Go has only one loop construct: `for`.
- Switch statements in Go are more powerful than in other languages because they don't require a `break` statement.
