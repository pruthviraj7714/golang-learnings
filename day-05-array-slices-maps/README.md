# Day 05 - Arrays, Slices, Maps

## What I Learned

- Arrays vs slices
- Slice operations (append, copy, slicing)
- Slice internals (length, capacity)
- Maps (key-value store)

## 📝 Notes

- Slices are the most used data structure in Go and share underlying memory.
- Maps are used to store key-value pairs.
- Arrays are rarely used in real apps
- slice is a reference type
- copy() is used to copy slices
- append() is used to add elements to a slice
- len() is used to get the length of a slice
- cap() is used to get the capacity of a slice
- make() is used to create a slice
- delete() is used to delete elements from a map
