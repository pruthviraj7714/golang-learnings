# 🚀 Day 01 – Basics of Go

## 📚 What I Learned

- 📦 Package declaration
- 🔢 Variables and constants
- 🧩 Data types
- 🖨️ `fmt.Println` and `fmt.Printf`

---

## 📝 Notes

- Go is a **statically typed** and **compiled** language
- Unlike Node.js, Go compiles directly to machine code
- Designed for **performance**, **simplicity**, and **concurrency**

---

## ⚔️ Go vs Node.js

| Concept        | 🟦 Go             | 🟩 Node.js        |
| -------------- | ----------------- | ----------------- |
| ⚙️ Execution   | Compiled          | Interpreted       |
| 🔄 Concurrency | Goroutines        | Event Loop        |
| 🧠 Memory Mgmt | Garbage Collector | Garbage Collector |
| 🔒 Typing      | Static            | Dynamic           |
| 🚀 Performance | Very High         | Medium            |
